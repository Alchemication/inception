# Inception!!!

Starting point for Node Apps

### Installation
  - install Node
  - clone this repo or download it
  - run `npm install`
  - run `npm start`
  - open http://localhost:3001 in the browser
  - have fun!!!

### Contains
  - Node server
  - AngularJS client
  - Some nice to have dependencies (Bootstrap, Underscore, Socket.io etc.)
